About ForbiddenBOT

https://www.facebook.com/forbiddenbot/

Forbidden BOT is made of a bunch of technologies, this is because I used the bot as a means to test my knowledge in these areas, and it made the project more complex than it could be.

The bot uses Angular 8 for the frontend, and Java Spring Boot for the backend, along with mysql. The backend is hosted for free on heroku, and the site is also hosted for free on Google. the images are stored on Amazon.

You can access the full project on GitHub.

Currently heroku's free server shuts down the bot every thirty minutes, donating $7.00 you can make this bot operational 24/7.
